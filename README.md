## Install

Step 1. Install <a href="https://nodejs.org/en/">Node</a> && <a href="https://www.npmjs.com/">NPM</a>（Node Package Manager）

Step 2. Download Framework

```shell
$ mkdir DaVinci
$ cd DaVinci
$ git clone https://github.com/qulongjun/DaVinci.git .
```

Step 3. Install Plugins
```shell
$ npm install
```

Step 4. Run Service
```shell
$ npm start
```

Step 5. Enjoy it
```shell
$ http://127.0.0.1:3001
```
